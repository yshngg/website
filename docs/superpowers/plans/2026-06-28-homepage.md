# Homepage Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform the current placeholder homepage into a professional landing page with hero, bio, skills, recent posts, featured articles, and open source projects.

**Architecture:** All content-driven — bio lives in `content/_index.md` (Section in Zola), projects in `data/projects.toml`, config data in `zola.toml` `[extra]`, template orchestrates everything.

**Tech Stack:** Zola (Tera templates), Oat CSS framework, no JS needed.

## Global Constraints

- All text content must be bilingual (EN/ZH) where marked in spec: hero title + bio
- Projects sourced from `data/projects.toml` only
- Skills, featured articles, tagline defined in `zola.toml` `[extra]`
- Use existing Oat utility classes only — no new CSS
- Template extends `base.html`
- Use `section` context (not `page`) in `templates/index.html`

### Task 3: Homepage Template

**Files:**
- Modify: `templates/index.html`

**Interfaces:**
- Consumes: `config.extra.tagline`, `config.extra.skills`, `config.extra.featured_articles`, `section.title`, `section.content`, `data.projects` (auto-loaded from `data/projects.toml`), `get_section("blog/_index.md")`
- Produces: rendered homepage HTML

- [ ] **Step 1: Rewrite `templates/index.html`**

```html
{% extends "base.html" %} {% block content %}
<section class="align-center" style="padding-block: var(--space-12)">
  <h1 style="margin-block: 0; font-size: var(--text-0)">{{ section.title }}</h1>
  <p class="text-light" style="font-size: var(--text-3); margin-block: var(--space-2) 0">
    {{ config.extra.tagline }}
  </p>
</section>

<section style="max-width: 48rem; margin-block-end: var(--space-12)">
  <h2>About</h2>
  {{ section.content | safe }}
</section>

<section style="margin-block-end: var(--space-12)">
  <h2>Skills</h2>
  {% for group in config.extra.skills %}
    <h3>{{ group.category }}</h3>
    <div class="hstack" style="margin-block-end: var(--space-4)">
      {% for skill in group.items %}
        <span class="badge">{{ skill }}</span>
      {% endfor %}
    </div>
  {% endfor %}
</section>

<section style="margin-block-end: var(--space-12)">
  <h2>Recent Posts</h2>
  <ul class="unstyled vstack">
    {% set blog = get_section(path="blog/_index.md") %}
    {% for post in blog.pages | slice(end=3) %}
      <li>
        <a href="{{ post.permalink | safe }}">{{ post.title }}</a>
        <span class="text-light">&middot; {{ post.date }}</span>
      </li>
    {% endfor %}
  </ul>
  <a href="{{ get_url(path='@/blog/_index.md') }}">View all blog posts &rarr;</a>
</section>

<section style="margin-block-end: var(--space-12)">
  <h2>Featured Articles</h2>
  <ul class="unstyled vstack">
    {% for path in config.extra.featured_articles %}
      {% set article = get_page(path=path) %}
      <li>
        <a href="{{ article.permalink | safe }}">{{ article.title }}</a>
        {% if article.extra.author %}
          <span class="text-light">&middot; {{ article.extra.author }}</span>
        {% endif %}
      </li>
    {% endfor %}
  </ul>
  <a href="{{ get_url(path='@/curated/_index.md') }}">View all curated articles &rarr;</a>
</section>

<section style="margin-block-end: var(--space-12)">
  <h2>Projects</h2>
  <ul class="unstyled vstack">
    {% for project in data.projects %}
      <li>
        <a href="{{ project.url }}" target="_blank">{{ project.name }}</a>
        <span class="text-light">&mdash; {{ project.description }}</span>
      </li>
    {% endfor %}
  </ul>
</section>
{% endblock content %}
```
